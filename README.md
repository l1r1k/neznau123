# MyMPT - Московский Приборостроительный Техникум

Мобильное приложение для студентов Московского приборостроительного техникума, позволяющее просматривать расписание занятий, звонки и другую полезную информацию.

## Описание

Приложение разработано для удобства студентов техникума. Оно предоставляет:
- Просмотр расписания на неделю
- Просмотр расписания на сегодня
- Информацию о звонках
- Настройки профиля

## Архитектура

Приложение следует принципам чистой архитектуры (Clean Architecture):

- **Слой представления (Presentation Layer)**: Содержит экраны и виджеты пользовательского интерфейса
- **Доменный слой (Domain Layer)**: Содержит бизнес-логику, сущности и интерфейсы репозиториев
- **Слой данных (Data Layer)**: Содержит реализации репозиториев и работу с API
- **Ядро (Core Layer)**: Содержит константы и утилиты, используемые по всему приложению

## Установка

1. Убедитесь, что у вас установлен Flutter SDK
2. Клонируйте репозиторий
3. Выполните `flutter pub get` для установки зависимостей
4. Запустите приложение с помощью `flutter run`

## Структура проекта

```
lib/
├── core/              # Ядро приложения (константы, утилиты)
├── data/              # Слой данных (модели, репозитории, сервисы)
├── domain/            # Доменный слой (сущности, интерфейсы, use-case'ы)
├── presentation/      # Слой представления (экраны, виджеты)
└── main.dart          # Точка входа в приложение
```
